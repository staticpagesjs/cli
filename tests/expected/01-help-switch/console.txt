Usage: staticpages [options]

Options:
  --version                      output the current cli version
  -c, --config <path>            path to a build configuration file
  -f, --from <package>           import 'cli' or 'default' from this package as
                                 the reader
  -a, --from-args <JSON-string>  arguments passed to reader; provide in JSON
                                 format
  -t, --to <package>             import 'cli' or 'default' from this package as
                                 the writer
  -b, --to-args <JSON-string>    arguments passed to writer; provide in JSON
                                 format
  -s, --controller <package>     controller that can process the input data
                                 before rendering
  -v, --variables <JSON-string>  additional object that will be passed to the
                                 controller as 'this'
  -h, --help                     display help for command
